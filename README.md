# Apex Overdrive

A browser-based 3D car racing game built with Three.js and cannon-es.

**Play it here: [https://apex-overdrive-tumo.netlify.app/](https://apex-overdrive-tumo.netlify.app/)**

## Development

```bash
npm install
npm run dev      # start Vite dev server (HMR)
npm run build    # production bundle
npm run preview  # serve the production build locally
```
